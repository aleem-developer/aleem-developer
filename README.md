## 🚀 Featured Project: A.Translator

**A.Translator** is a smart, multilingual PDF-to-Word converter  
✅ Built in Python using Google Translate API  
✅ Translates full PDFs into multiple languages  
✅ Supports right-to-left languages (Urdu, Arabic, etc.)  
✅ Generates formatted DOCX with smart bullet alignment  
✅ Built with async logic, custom GUI, and splash screen
